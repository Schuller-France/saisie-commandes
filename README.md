# Saisie de commande commerciale

Application statique de saisie de commande pour commerciaux terrain.

## Important

Le fichier `outputs/data.js` contient la liste clients et le tarif. Ce dépôt doit rester privé et ne doit pas être publié sur un hébergement public sans authentification.

## Lancer l'application

Ouvrir `outputs/index.html` dans un navigateur.
